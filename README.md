# Spring Boot 3.1 pet project
Spring Boot 3.1 기준 프로젝트.


## Features
- REST API
- AOP logging
- Spring security implementation (lambda dsl)
- JWT authentication with RSA key pair
- Refresh token
- Exception handling
- Http client with Retrofit2


## Technologies
- Spring Boot 3.1
- Spring Web
- Spring Data JPA
- Spring Security
- JSON Web Token (JWT)
- H2 Database
- Retrofit2
